| config/processes.json → index.js | caseNumber values from config file are used directly in file paths and as browser input |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Tampering | validateCaseNumber() + file path construction | mitigate | Validate caseNumber matches /^[a-f0-9]{8,}$/ before using in state/<caseNumber>.json path — prevents path traversal (e.g., "../../etc/passwd" as caseNumber) |
| T-01-02 | Availability | saveState() atomic write | mitigate | Write to <target>.tmp then renameSync — prevents corrupt state file on crash mid-write (STATE-02) |
| T-01-03 | Information Disclosure | stdout output | accept | movimentação text is public court record (consulta pública) — no PII beyond what is already public |
| T-01-04 | Denial of Service | sequential HTTP to government site | accept | No parallel requests, delay+jitter between cases — low traffic profile; government scraping risk is low |
</threat_model>
